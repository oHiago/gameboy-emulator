# 🎮 Emulador de Game Boy em C++

Um emulador do Game Boy (DMG) construído inteiramente do zero em C++, sem uso de bibliotecas de emulação prontas, com o objetivo de aprender profundamente sobre arquitetura de computadores, sistemas de baixo nível e emulação de hardware.

Este não é um projeto focado em performance ou compatibilidade total — é, antes de tudo, um projeto de **estudo**.

## 🎯 Objetivo

O objetivo deste projeto é construir um emulador funcional do Game Boy original (DMG), do absoluto zero, em C++, cobrindo desde a CPU (Sharp LR35902) até vídeo (PPU), áudio (APU), timers e interrupções — entendendo e implementando manualmente cada componente do hardware.

## 📚 Status do Projeto

- [ ] Estrutura inicial do projeto e sistema de build
- [ ] Leitura de ROM / Cartridge (header, MBCs)
- [ ] Bus (roteamento de memória)
- [ ] CPU (registradores, flags, ciclo fetch-decode-execute)
- [ ] Conjunto de instruções (opcodes)
- [ ] Interrupções (IE/IF)
- [ ] Timer (DIV, TIMA, TMA, TAC)
- [ ] PPU (vídeo — scanlines, tiles, sprites)
- [ ] Joypad (entrada)
- [ ] APU (áudio — 4 canais)
- [ ] Testes com ROMs de validação (Blargg, Mooneye)
- [ ] Primeiro jogo comercial rodando de ponta a ponta 🎉
